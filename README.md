LojaTerco - Fullstack (React + FastAPI + PostgreSQL)

Conteúdo:
- frontend/ (React + Vite)
- backend_python/ (FastAPI + asyncpg)
- migrations/01_init.sql
- docker-compose.yml

Instruções rápidas:
1. Configure .env se quiser mudar credenciais (opcional).
2. Suba com Docker: `docker compose up --build`
3. A API estará em http://localhost:4000
4. Frontend (Vite) estará em http://localhost:5173 (ou sirva o build)
